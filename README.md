# 明日あさ走ってもいい？チェッカー

「無理をしてでも走ってしまう」真面目なランナーや、不規則な勤務が続く医療従事者のための、寝る前専用セルフケア診断Webアプリ。

## ⚙️ 判定システム
自分の意志だけでは「走らなきゃ」と無理をしてしまう時の、頼れる客観的なブレーキとして機能します。

### 1. 絶対条件（一発アウト）
以下の項目が1つでもある場合、翌朝は**強制休養**となります。
- 「走りたくない」という心のサイン（重要変更）
- 体調不良（発熱・喉の痛みなど）
- 5時間以下の睡眠不足
- 「走らなければいけない」という精神的義務感
- 雨予報

### 2. 累積条件（合わせ技アウト）
以下の項目のうち、**3項目以上**が重なった場合は**強制休養**となります。
- ロング日勤後、眠気、ダルさ
- 足や関節の軽い違和感（調整）
- 日中の水分不足
